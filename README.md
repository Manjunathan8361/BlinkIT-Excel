🛒 Blinkit Sales Performance Dashboard (Excel)
📌 Project Overview

This project is an interactive Sales Analytics Dashboard built in Microsoft Excel to analyze Blinkit’s grocery sales performance across multiple dimensions including:

Outlet Establishment Year

Outlet Type

Outlet Size

Product Category

Location Tier

Item Fat Content

Year-wise Sales & Ratings

The goal of this project is to extract actionable business insights from raw transactional data and present them in a structured, decision-ready dashboard.

🎯 Business Objectives

Identify high-performing product categories

Analyze sales trends year-wise

Compare performance by outlet size and type

Understand tier-wise location impact

Evaluate customer rating trends

Detect contribution of item fat content to revenue

🧹 Data Cleaning Process

Before building the dashboard:

Separated combined fields (Year + Outlet ID)

Removed blank values

Standardized year format

Eliminated duplicate categories

Cleared Pivot Cache to avoid ghost values

Ensured numeric consistency

This ensured accurate aggregations and slicer behavior.

📌 Key Insights

Certain product categories like Fruits & Vegetables and Snack Foods drive higher revenue.

Tier 3 outlets show strong contribution to total sales.

Medium and Small outlet sizes dominate distribution.

Sales show upward movement in recent establishment years.

Rating trends fluctuate but remain within a tight range (3.9–4.0).
